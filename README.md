**Capybara** is a desktop app that lets you run an army of Claude Code sessions at once. Open a project, spawn agents, and watch them all work in parallel. 

Capybaras build Capybara!

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed and authenticated

## Install & Run

```bash
git clone https://github.com/evancdev/capybara.git
cd capybara
pnpm install
pnpm dev
```

That's it. A window opens, you pick a folder, and you're off.
